# 餘額式記帳 APP
針對手機製作的記帳 APP，和一般記帳軟體不同的是，是用"目標"的方式來進行記帳，以便於做計劃(e.g. 本月花費目標等)

# 使用技術
主要使用 Javascript 搭配 Bootstrap 5，並使用 Capacitor 轉 APP
